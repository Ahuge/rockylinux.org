The Rocky Linux landing page, built using [Gatsby](https://gatsbyjs.com) and [TailwindCSS](https://tailwindcss.com).

## Start Development Server

1. Install dependencies via NPM.

```bash
npm i
```

2. Start the development server.

```bash
gatsby develop
```
